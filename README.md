# Statistical Data Analysis Projects

This repository contains three projects developed as part of a Statistical Data Analysis course.  
Each project focuses on a different core methodology used in data science and applied statistics.

The projects are implemented in Python using standard scientific libraries and are documented through Jupyter notebooks and accompanying reports.

### **Project 1 ‚Äì Maximum Likelihood Parameter Estimation**
**Topic:** Gaussian model with unknown parameters  
**Reference:** Cowan, Section 6.8

**Objective:**

---

### **Project 2 ‚Äì Least Squares Method (LSM)**
**Topic:** Parameter estimation and model comparison  
**Reference:** Cowan, Chapter 7

**Objective:**
Fit experimental data using different functional models and determine which model best describes the data using the Pearson chi-square test.

**Key concepts:**
- Weighted least squares
- Analytical vs numerical minimization
- Chi-square statistic
- Reduced chi-square and p-value
- Model comparison

**Models tested:**
- Power-law model  
- Quadratic polynomial  
- Linear + exponential model  

**Deliverables:**
- Jupyter Notebook
- PDF report
- CSV data file

---

### **Project 3 ‚Äì Time Series Analysis and Audio Denoising**
**Topic:** Spectral analysis of audio signals

**Objective:**
Clean a noisy audio recording of a Mozart excerpt using frequency-domain techniques.

**Key concepts:**
- Time-domain vs frequency-domain representation
- Fast Fourier Transform (FFT) and inverse FFT
- Power Spectral Density (PSD)
- High-pass filtering
- Spectrogram analysis

**Processing steps:**
- Noise identification via FFT and PSD
- Filter design and testing
- Time‚Äìfrequency validation
- Audio reconstruction

**Deliverables:**
- Jupyter Notebook
- PDF report
- Original and filtered WAV files
